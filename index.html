PYTHON ML MODELS

TSNE-HDBSCAN

Sklearn

To cluster and visualize data, we often need to apply dimension reduction first. PCA can perform poorly when the number of features is large due to curse of dimensionaity effect, so tSNE can be a choice with its non-Euclidean tooling. If cluster patterns are complex k-Means can fail, and procedural techniques like hDBSCAN can perform much better. See this repo, where each of these techniques is applied on a toy dataset

LSTM-SENTIMENT

Pytorch

In this exercise I tried to productionize LSTM model, which was available from a Github repo here . Advantage of LSTM vs RNN's is that they capture longer-run effects. After training the model on a Google Colab GPU, I saved the weights and use statically during inference. The app is served with PM2 process manager and can be tested by sending it POST request at the URL, specified in this repo

TAG-PREDICTOR

Sklearn

The aim is to construct tag predictor on Stackoverflow posts using a linear classifier. NLTK library is used to filter out stop words and tokenize text at word level. The text is vectorized using BoW method. TF-IDF doesn't seem to add much value in this setting. Logistic regression is used as a base model, and OneVsRestClassifier is trained on top of them. To find out more, check out this repo

TIME-SERIES

Pyflux

The project is a brief experiment with time series modeling on a medium-sized dataset. The features evidently had autoregressive components, requiring ARIMA. Differentiation was not used since the series were mean-stable, that is no trends, and no evident seasonality either. There are 4 predicted series that show moderate variation in volatility. To find out more, check out this repo

FLASK-RESNET

Pytorch

This is an image classification built on RESNET model on Pytorch. The model is pre-trained using ImageNet dataset. Being based on this arxiv paper, this network uses additional blocks that are generated skipping some layers of base network. The code is taken from official Pytorch tutorials, while my minor contribution was to deploy it on own dev-server using Flask/uWSGI. To find out more, check out this repo

CNN-fMNIST

Pytorch

This is an image classification model on Pytorch using a CNN trained on fashion-MNIST image dataset. There are eight layers defined in this CNN architecture, including two sequences of Conv2d, BatchNorm2d, MaxPool2d and a fully-connected layer at the end. The task was an assignment as part of IBM course on Introduction to Pytorch on Coursera. To find out more, check out this repo

RNN-CLASSIFICATION

Pytorch

This is an character-level RNN on Pytorch, which is trained on a corpus of names from 18 languages and tries to predict which languages the name comes from. The code for model was taken from this Pytorch official tutorials . RNNs are powerful models that capture the sequence or order-of-flow effect in data. To find out more, check out this repo
