MY STACK

MACHINE LEARNING

Pytorch is the main framework I use to prototype machine learning models, while Keras, Tensorflow are secondary. The models are served with Flask, uWSGI

BACKEND-END

On the backend, I prefer to use Node.js as it is a powerful language, scalable and has a rich package library. For databases, I have familiarity with MongoDB and Redis

FRONT-END

On the front-side, I use Vue.js, but my exposure to it is not extensive. For component library, I use Bootstrap-vue

PIZZERIA

Web App

This generic design template was developed for the food category. Shop owners can view all their incoming orders and edit their products from the web dashboard. Session and cart data is stored server-side in Redis and Mongo databases, respectively. Click here to find out more

FACEBOOK BOT

Messenger App

This messenger bot was designed to address small flower shop owners that can add their merchandize in real-time through web UI, but receive orders on Facebook. Client orders can be viewed on the admin dashboard. Since client data is retained, shops can re-target their clients. Re-factor in progress on this app presently

TELEGRAM BOT

Messenger App

This messenger bot is for small barber shops, managing appointments from Telegram. Other solutions, eg web forms, take more time and steps, irritating clients. Messenger apps handles appointment task more neatly and allows for personalized customer engagement. Bot is available at this link
